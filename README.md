# Финансовая Олимпиада — Платформа Бизнес-Симуляции

## Идея проекта

Образовательная платформа для детей 11–18 лет (5–11 класс), где участники соревнуются в управлении виртуальным бизнесом. Вдохновлена форматом международных финансовых олимпиад: первый этап — теоретические задачи, второй — управление компанией в симуляции.

Ориентация: русскоязычная аудитория, российский рынок.

---

## Формат олимпиады

### Этап 1 — Теоретические задачи
Решение финансово-экономических задач по возрасту.

### Этап 2 — Симуляция бизнеса
Управление виртуальной компанией в течение 6 игровых месяцев (ходов).

**Режим:** пошаговый (каждый ход = 1 месяц). Архитектурно движок не зависит от триггера, что позволяет в будущем перейти на режим реального времени с ускорением без переписывания экономической логики.

**Игрок:** одиночный (командный режим — в дорожной карте).

---

## Игровой процесс

Каждый ход игрок принимает решения:
- Установить цены на товары/услуги
- Выбрать объём рекламы
- Нанять или уволить сотрудников

После хода — результаты: выручка, расходы, прибыль, удовлетворённость клиентов.

Происходят **случайные события**: сезонность, открытие конкурента, праздники, кризис.

### Сложность по возрасту

| Возраст | Класс | Решений за ход | События | Финансовые инструменты |
|---|---|---|---|---|
| 11–13 | 5–7 кл | 3 простых | редко, мягкие | выручка / расходы |
| 14–16 | 8–10 кл | 5–6 | чаще, жёстче | + кредит, налог, аренда |
| 17–18 | 11 кл | 8–10 | реалистичные | + инвестиции, конкуренты, найм |

---

## Шаблоны бизнеса

Игрок выбирает **архетип бизнеса** и кастомизирует его (название, продукт, аудитория).

MVP-шаблоны: магазин, кафе.
Дорожная карта: IT-сервис, школа, агро-ферма и другие.

## Результат для участника

По итогам симуляции ребёнок получает:

### Бизнес-архетип
| Архетип | Описание |
|---|---|
| Визионер | Много рискует, инвестирует в рост |
| Финансовый снайпер | Считает каждую копейку, высокая маржа |
| Маркетолог от бога | Клиентская любовь ценой высоких расходов |
| Антикризисный менеджер | Умеет выруливать из трудных ситуаций |
| Стратег | Сбалансированный стабильный рост |

### Персональный разбор
- Что получалось хорошо (с конкретными цифрами из игры)
- Над чем стоит поработать
- Рекомендации: книги, курсы, видео — по возрасту

### Рекомендации по возрасту
- **11–13:** простой язык, мультики, Кийосаки (детская версия), «попробуй продать что-нибудь»
- **14–16:** маржинальность, Наполеон Хилл, курсы ЦБ РФ по финграмотности
- **17–18:** кейсы реальных компаний, Питер Тиль «От нуля к единице», профориентация

### Сертификат
Скачиваемый PDF с результатами и местом в лидерборде.

## Монетизация и спонсоры (B2B2C)

### Четыре уровня

| Уровень | Кто | Что получает |
|---|---|---|
| Freemium | Дети, школы | Стандартные шаблоны бизнеса, открытые олимпиады — бесплатно |
| Sponsor Tier | Корпорации (Магнит и др.) | Брендированный шаблон, свой турнир, аналитика — от 500к руб/год |
| Государственный | МинОбр, регионы | White-label платформы, интеграция в учебную программу, сертификаты |
| Вузовский | МФТИ, ВШЭ, МГИМО и др. | Попадание в персональные рекомендации по итогам игры — от 200к руб/год |

### Что получает корпоративный спонсор
- Брендированная симуляция (дети управляют «магазином Магнит» с реальными категориями товаров)
- Именной турнир («Кубок Магнит 2026») с призами от компании
- Дашборд аналитики: сколько детей играло, какие решения принимали
- Кастомные сценарии (спонсор добавляет свои кейсы)

### Архитектурная основа под спонсоров
```
Organization (спонсор)
  └── BusinessTemplate (шаблон бизнеса)
        ├── branding (логотип, цвета, название)
        ├── product_catalog (товары / услуги)
        ├── scenarios (кастомные задачи)
        └── metrics (критерии победы)
  └── Tournament (именная олимпиада)
```

- **Multi-tenancy** — каждый спонсор изолирован, видит только своих участников
- **Branding слой** — CSS переменные + JSON-конфиг, не хардкод
- **Self-service кабинет спонсора** — загрузить логотип, создать турнир, смотреть статистику

## Профориентация через игровое поведение (8–11 класс)

В процессе игры система анализирует паттерны принятия решений и определяет когнитивный профиль — не тестом, а поведением.

### Поведенческие профили

| Профиль | Сигналы в игре |
|---|---|
| Физ-мат / аналитик | Точная оптимизация цен, максимизация прибыли, игнорирует "мягкие" факторы |
| Экономист / стратег | Балансирует рост и стабильность, хорошо реагирует на кризисы |
| Маркетолог / гуманитарий | Инвестирует в рекламу, ставит на удовлетворённость клиентов |
| Предприниматель | Кредиты, агрессивный найм, высокая дисперсия результатов |

### Рекомендации по результату

| Уровень + профиль | Рекомендации |
|---|---|
| Топ + физ-мат | МФТИ, мехмат МГУ, ВШЭ прикладная математика |
| Топ + экономика | ВШЭ, РЭА Плеханова, МГИМО |
| Топ + гуманитарий | МГУ (журфак, соцфак), МГИМО |
| Средний уровень | Ведущие региональные вузы по профилю |
| Базовый уровень | Гос. вузы, прикладные специальности |

Профилирование — MVP-1 (после базовой игры). Архитектурно: каждое решение логируется, профиль = вектор метрик, считается на бэкенде.

## Интеграция VK Mini Apps / MAX

**MAX** (100 млн зарег., 70 млн DAU) — российский super-app, обязателен на всех смартфонах с сентября 2025.

Платформа для интеграции: **VK Mini Apps** (45 млн MAU, зрелый JS SDK).

- Игра запускается прямо в VK/MAX — без установки приложения
- Авторизация через VK (у большинства школьников уже есть аккаунт)
- Шеринг результатов / архетипа в ленту VK → виральность
- Уведомления через VK

## MVP — первая версия

**Цель:** живая демо для МинОбра или первого спонсора.

**Входит в MVP:**
- Регистрация (email, Google, VK) с указанием возраста
- 2 шаблона бизнеса: магазин и кафе
- Симуляция на 6 ходов (пошаговая)
- Таблица лидеров
- Персональный результат + архетип + сертификат PDF
- Базовая аналитика для администратора

**Не входит в MVP (дорожная карта):**
- Кабинет спонсора (архитектура заложена, UI — позже)
- Командный режим
- Этап 1 (теоретические задачи)
- Мобильное приложение
- Реальное время / ускорение

## Стратегия государственных грантов

Платформа архитектурно выровнена под требования грантодателей:

| Источник | Программа | Сумма | Ключевое требование |
|---|---|---|---|
| ФСИ | "Старт-1" | до 4 млн руб | IT-продукт, команда |
| Сколково | Грант резидента | до 5 млн руб | Инновация, рынок |
| ЦБ РФ | Финансовая грамотность | варьируется | Alignment с нацстратегией-2030 |
| Росмолодёжь | Молодёжные проекты | до 3 млн руб | Целевая аудитория: молодёжь |
| ПФКИ | Образование | до 10 млн руб | Культурная / образовательная ценность |

**Обязательные условия:**
- Соответствие ФГОС (финансовая грамотность обязательна в школах с 2024)
- Отечественный стек (VK, российский хостинг)
- Измеримый образовательный результат (до/после)
- Охват всех регионов, бесплатный доступ для детей

## Дорожная карта

1. **MVP-0** — веб, одиночный, 2 шаблона, лидерборд, архетип, сертификат
2. **MVP-1** — поведенческое профилирование + рекомендации вузов (8–11 кл)

5. Командный режим
6. Интеграция с МинОбр / региональные олимпиады
7. Этап 1 (теоретические задачи)
8. Мобильное приложение (iOS + Android)
9. Режим реального времени с ускорением
